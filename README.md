# LotoFácil Intelligence Lab (LIL)

Plataforma pessoal de pesquisa estatística e análise avançada da Lotofácil.

## Objetivos

- Analisar dezenas utilizando múltiplas métricas estatísticas
- Executar estratégias explicáveis
- Comparar estratégias
- Descobrir padrões estatísticos
- Evoluir continuamente através de versões

## Tecnologias

- Vue 3
- Vite
- Pinia
- JSON
